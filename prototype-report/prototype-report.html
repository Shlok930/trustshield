Feature Inventory

Wallet Lookup

  • Ethereum wallet address input and validation.
  • Backend API route for wallet scanning.
  • Vite proxy support for local development.
  • Production backend URL support for Vercel + Render deployment.

Etherscan Data

  • Live balance lookup from Etherscan.
  • Latest normal transaction records.
  • Latest token transfer records.
  • Transaction hashes, from/to addresses, values, dates, and token metadata.

Risk Dashboard

  • Wallet overview with balance, activity, age, and status.
  • Heuristic risk score and trust score.
  • AI-style reputation, anomaly, sybil, whale, fraud, and MEV labels.
  • Behavioral analysis and suggested security actions.

Analytics Panels

  • Transaction analytics with incoming/outgoing ratios.
  • Gas spend and largest transaction metrics.
  • Recent transaction timeline.
  • Token holdings and contract interaction summaries.

Security Modules

  • Approval risk panel.
  • Scam and phishing risk indicators.
  • Contract interaction footprint.
  • Recommended actions and high-risk warnings.

System Notes

  • Raw Etherscan records are actual API data.
  • AI/risk sections are app-generated estimates, not verified Etherscan facts.
  • Backend rate-limits Etherscan calls to avoid the 3/sec error.
  • Browser console logs the full wallet API response for inspection.
